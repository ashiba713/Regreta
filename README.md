Regreta

Career Regret Engine is a **Streamlit-based web app** that helps users reflect on their career decisions and visualize possible regrets. The app lets users input details about their career choices (like the field, years of experience, satisfaction, risk tolerance, etc.) and then analyzes these inputs to generate a *career regret index* and personalized insights.


Features

- Interactive Streamlit UI.
- User inputs for:
  - Current career field
  - Years of experience
  - Job satisfaction level
  - Future goals
  - Regret sensitivity
- Visualization of regret scores and scenario comparison.
- Suggestions to reduce future career regret.

 Getting Started

 Requirements

Make sure you have:

- Python 3.7+
- Streamlit
- Pandas, NumPy
- Matplotlib or Plotly (for charts)
